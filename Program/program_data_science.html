Data Science

Program Data Science: Menguasai Ilmu dari Data untuk Membuat Keputusan yang Lebih Baik

Melalui kurikulum komprehensif ini, kamu akan mempelajari cara mengolah data dari nol, melakukan analisis statistik, hingga membangun model prediksi menggunakan machine learning. Di akhir program, kamu akan memiliki keahlian untuk memecahkan masalah bisnis nyata menggunakan data dan mampu memvisualisasikan hasilnya dengan cara yang mudah dipahami.

Di akhir program, kamu akan memiliki keahlian untuk memecahkan masalah bisnis nyata menggunakan data dan mampu memvisualisasikan hasilnya dengan cara yang mudah dipahami.

Yang Akan Kamu Pelajari:

  • Data Preparation & Cleaning:Teknik untuk menyiapkan dan membersihkan data agar siap dianalisis.
  • Exploratory Data Analysis:Cara mengeksplorasi data untuk menemukan pola dan tren penting.
  • Machine Learning:Penerapan algoritma supervised dan unsupervised learning untuk membuat model prediktif.
  • Data Visualization:Membuat visualisasi data yang efektif untuk menyampaikan hasil analisis secara jelas dan menarik.

Silabus

Durasi: 14 Minggu

Mode Belajar: Kelas Online, Video On-Demand, Diskusi Mentor

Proyek: 2 Proyek Mini + 1 Proyek Akhir

  • Pengantar Data Science: Apa itu Data Science dan aplikasi dalam berbagai industri
  • Instalasi & Setup: Jupyter Notebook, Anaconda, Python, R, dan alat-alat analisis lainnya
  • Pemrograman Dasar Python: Variabel, tipe data, control flow, dan fungsi
  • Pengenalan ke Statistik Dasar: Mean, median, mode, standar deviasi, distribusi probabilitas
  • Sumber Data: Mengakses data dari file CSV, Excel, database, dan API
  • Data Wrangling: Teknik untuk menggabungkan, mengubah, dan menyaring data
  • Pembersihan Data: Menangani missing values, duplikasi, dan data yang tidak konsisten
  • Exploratory Data Analysis (EDA): Visualisasi awal untuk memahami pola dan outliers
  • Analisis Statistik Lanjut: Korelasi, regresi, dan analisis distribusi
  • Pandas & Numpy: Library Python untuk analisis data yang lebih mendalam
  • Matplotlib & Seaborn: Visualisasi data dengan grafik dan diagram interaktif
  • Dashboarding dengan Tableau/Power BI: Membuat laporan interaktif dan dashboard
  • Pengenalan ke Machine Learning: Supervised dan unsupervised learning
  • Algoritma Supervised Learning: Linear regression, logistic regression, decision trees
  • Evaluasi Model: Cross-validation, confusion matrix, precision, recall, F1-score
  • Pengantar Unsupervised Learning: Clustering dengan K-Means dan hierarchical clustering
  • Algoritma Lanjut: Random forest, gradient boosting, support vector machines (SVM)
  • Feature Engineering: Teknik memilih dan membuat fitur yang relevan
  • Hyperparameter Tuning: Optimasi model machine learning
  • Proyek Mini: Membangun model machine learning dengan dataset nyata
  • Pengantar NLP: Analisis teks, tokenisasi, stemming, dan lemmatization
  • Sentiment Analysis: Menganalisis sentimen dari teks atau media sosial
  • Big Data: Pengenalan Hadoop, Spark, dan pengolahan data skala besar
  • Proyek Mini: Analisis teks atau implementasi big data dalam studi kasus nyata
  • Proyek Akhir: Membangun proyek data science dari end-to-end, mulai dari pengumpulan data hingga presentasi hasil
  • Membuat Portofolio: Menyusun hasil proyek dalam portofolio profesional
  • Persiapan Karier: Tips membuat CV, interview, dan simulasi interview teknis
